# Brandname Generator

## Overview
The Brandname Generator is a simple application that generates creative brand name suggestions based on user input. Users can enter their city name and their pet's name, and the application will provide a list of potential brand names.

## Files in the Project
- **brandname_Generator.py**: This file contains the backend logic for generating brand names based on user input. It prompts the user for a city name and a pet name, then generates a list of brand name suggestions.
  
- **index.html**: This file serves as the user interface for the Brandname Generator. It includes a form for users to input their city name and pet name, and a section to display the generated brand name suggestions.

- **README.md**: This documentation file explains how to set up and use the Brandname Generator.

## How to Use
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to access the user interface.
3. Enter your city name and pet name in the provided input fields.
4. Click the button to generate brand name suggestions.
5. View the generated suggestions displayed on the page.

## Requirements
- A modern web browser to run the `index.html` file.
- No additional libraries or frameworks are required for this project.
